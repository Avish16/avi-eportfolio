Program overview

This section explains what I learned in the M.S. Applied Data Science program, how my projects connect to those outcomes, and how I applied those skills in research and internship work.

Program learning outcomes

My graduate training at Syracuse University focused on applied data science that is technically strong and practically useful. Based on my coursework and project work, I summarize the core learning outcomes as follows.

  1. Data and statistical foundations: clean, analyze, and interpret real-world datasets using sound statistical reasoning and exploratory analysis.
  2. Machine learning and deep learning practice: build, evaluate, and improve models for classification, regression, forecasting, and NLP tasks.
  3. End-to-end system design: move from raw data to usable outputs through reproducible pipelines, validation checks, and deployment-ready workflows.
  4. Communication and decision support: present technical work clearly to non-experts using dashboards, visualizations, and structured explanations.
  5. Responsible and human-centered AI: account for reliability, transparency, privacy, and failure cases, especially in healthcare and safety-related use cases.

High-level project summary

  • MediExplain: A human-centered medical AI system using LLMs, RAG, ChromaDB, and Streamlit to explain complex medical content for patients and caregivers with structured and validated outputs.
  • Driver Drowsiness Detection: A deep learning and computer vision project using CNN-based eye-state modeling, temporal aggregation, and OpenCV deployment for live webcam and video monitoring.
  • DataThon26, Syracuse spatial analytics: A team project that links code violations and property assessment data, builds grid-level risk features, and delivers interactive Folium maps for city-level insights.
  • Bitcoin Flash Crash Prediction: A minute-level event prediction workflow combining market features with Reddit sentiment and sequence modeling to classify flash crash events.
  • Northwind Data Warehouse and BI: Snowflake warehouse modeling, SQL ETL pipelines, and Power BI dashboards for operational KPIs and business insight generation.
  • Forecasting Energy Consumption: Time-series modeling on 50M+ records using ARIMA and XGBoost, with an R Shiny dashboard for planning scenarios.

This combination of coursework, projects, and professional experience helped me build a practical skill set in applied data science. It also strengthened my ability to deliver work that is technically accurate, reproducible, and useful for real users.
